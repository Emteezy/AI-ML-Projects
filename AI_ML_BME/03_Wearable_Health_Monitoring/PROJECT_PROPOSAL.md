## ğŸ“Š Project Overview

### What It Does

A complete IoT + ML system for real-time health monitoring:

- **Edge Device**: Raspberry Pi with health sensors (HR, SpO2, Motion)
- **Edge ML**: TensorFlow Lite models for on-device inference
- **Data Streaming**: MQTT protocol for real-time data transmission
- **Cloud Backend**: FastAPI for data aggregation and analysis
- **Dashboard**: Streamlit for real-time visualization and alerts

### Technical Stack

- **Hardware**: Raspberry Pi 4, Sensors (MAX30102, MPU6050)
- **Edge ML**: TensorFlow Lite
- **IoT Protocol**: MQTT (Eclipse Mosquitto)
- **Backend**: FastAPI (Python)
- **Frontend**: Streamlit
- **Database**: PostgreSQL / InfluxDB (time-series)
- **Deployment**: Docker, Docker Compose

### Key Features

1. **Sensor Integration**
   - Heart Rate monitoring
   - Pulse Oximetry (SpO2)
   - Accelerometer/Gyroscope (Activity tracking)
   - Temperature monitoring

2. **Edge Machine Learning**
