## ðŸ“Š Project Overview

### What It Does

Classify ECG signals into different arrhythmia types:
- **Normal Sinus Rhythm**
- **Atrial Fibrillation (AFib)**
- **Premature Ventricular Contraction (PVC)**
- **Supraventricular Tachycardia (SVT)**
- **Ventricular Tachycardia (VT)**

### Technical Stack

- **Deep Learning**: PyTorch (LSTM, Transformer)
- **Signal Processing**: SciPy, NumPy, Wavelets
- **API**: FastAPI (REST API)
- **Frontend**: Streamlit (Interactive UI)
- **Deployment**: Docker
- **Dataset**: MIT-BIH Arrhythmia Database (PhysioNet)

### Key Features

1. **Signal Preprocessing**
   - Baseline correction
   - Noise filtering (bandpass, notch)
   - QRS detection
   - Feature extraction

2. **Deep Learning Models**

## ðŸ“š Dataset: MIT-BIH Arrhythmia Database

### Why This Dataset?

- **Industry Standard**: Most widely used ECG dataset
- **Free & Accessible**: Available on PhysioNet
- **Well-Documented**: Extensive annotations
- **Real Clinical Data**: 48 half-hour recordings
- **Multiple Arrhythmias**: Various cardiac conditions

### Dataset Details

- **Source**: PhysioNet
- **Size**: ~48 recordings, 30 minutes each
- **Sampling Rate**: 360 Hz
- **Leads**: 2 leads (MLII, V1)
- **Annotations**: Expert-annotated beat labels

### Download Instructions

1. Create account on [PhysioNet](https://physionet.org/)
2. Download [MIT-BIH Arrhythmia Database](https://physionet.org/content/mitdb/1.0.0/)
3. Extract to `data/mitdb/` directory
