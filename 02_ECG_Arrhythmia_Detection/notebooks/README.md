# Notebooks

This directory contains Jupyter notebooks for data exploration and analysis.

## Notebooks

- `data_exploration.ipynb`: Explore the MIT-BIH Arrhythmia Database, visualize ECG signals, and analyze annotations
- `signal_analysis.ipynb`: Detailed signal processing analysis and feature extraction

## Usage

1. Make sure you have Jupyter installed:
   ```bash
   pip install jupyter
   ```

2. Start Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

3. Navigate to this directory and open a notebook

## Note

To use the notebooks, ensure:
- The MIT-BIH dataset is downloaded to `../data/mitdb/`
- All dependencies are installed (see `../requirements.txt`)
- The project structure is set up correctly
