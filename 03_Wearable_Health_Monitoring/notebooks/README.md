# Notebooks

This directory contains Jupyter notebooks for data exploration and model training.

## Notebooks

- `data_exploration.ipynb`: Explore sensor data, visualize health metrics
- `model_training.ipynb`: Train edge ML models for health classification and activity recognition

## Usage

1. Make sure you have Jupyter installed:
   ```bash
   pip install jupyter
   ```

2. Start Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

3. Navigate to this directory and open a notebook

## Note

To use the notebooks, ensure:
- All dependencies are installed (see `../requirements.txt`)
- Sensor data is available in `../data/` (or use simulated data)
- The project structure is set up correctly
